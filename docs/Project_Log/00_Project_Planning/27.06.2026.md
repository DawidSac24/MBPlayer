## Project Overview
I would like to make a portable dap with a modular DAC. I would make a base PCB, with all the necessary components like: 
- the microcontroller
- the SD card connector
- the USB C port (for charging and maybe file transfer)
- the power input
- the m.2 connector - ensuring communication between the main PCB and the DAC module.
- the audio output - the M.2 port may return the audio - to centralize the functionality putting the jack on the main PCB side.

### Different versions
The Project could have multiple versions:
- A default one, without the M.2 connector having everything on one board inside one case.
- A modular closed case with the M.2 connector. This would allow to change the DAC while having the robustness of a full case. When changing the DAC, the case would have to be unscrewed.
- A modular 'open' case, with the M.2 connector visible. This would allow for a fast and easy DAC swap but may have some physical vulnerabilities/flaws.
