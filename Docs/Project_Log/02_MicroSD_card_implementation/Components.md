## The micro SD Card Breakout card
I got a breakout board because I wanted to first Test every part individually before making the PCB's. I bought the Adafruit 4682 since that's the only one with SDIO that I could find on amazon and wanted to get one as quickly as possible. I also wanted the SD card reader to be SDIO compatible to maximise it's bandwidth speed.
